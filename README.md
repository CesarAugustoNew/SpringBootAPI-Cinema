# 🎬 SpringBootAPI-Cinema

API RESTful desenvolvida em **Java** utilizando **Spring Boot** para gerenciamento de um catálogo de filmes. O projeto foi criado com o objetivo de aplicar os principais conceitos de desenvolvimento Backend, arquitetura em camadas e boas práticas na construção de APIs REST.

---

## 📖 Sobre o Projeto

Esta aplicação permite realizar operações completas de **CRUD** sobre filmes, disponibilizando endpoints para cadastro, consulta, atualização e exclusão de registros.

O projeto foi desenvolvido como prática de desenvolvimento Backend utilizando o ecossistema Spring.

---

## 🚀 Funcionalidades

- ✅ Autenticação com JWT
- ✅ Login de usuários
- ✅ Controle de acesso com Spring Security
- ✅ Cadastro de filmes
- ✅ Listagem de filmes
- ✅ Busca de filme por ID
- ✅ Atualização de filmes
- ✅ Exclusão de filmes
- ✅ Persistência em PostgreSQL
- ✅ Documentação com Swagger/OpenAPI
